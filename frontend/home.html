About

This was made with the intent of learning to create a fullstack application. The frontend was created using HTML, CSS and Javascript, while the backend was created using the Django Rest Framework.

This project is by no means perfect, but it served as an effective way for me to learn the aforementioned programming languages/frameworks. I have certainly learnt a lot from this project, and I am excited to look back at it in the future to see my improvement.
